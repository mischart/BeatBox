# BeatBox

## Projektbeschreibung
Eine Webanwendung zur Erstellung von Beats, die im Rahmen der Lehrveranstaltung  „Rich-Media-Anwendungen” umgesetzt wurde.
[Link zur lauffähigen Anwendung](http://hultaaj.bplaced.net/BeatBox/)

## Technologien
